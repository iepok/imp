# imp

**imp** — short for **implement** — is a CLI-first goal tracking and productivity system.

The core philosophy: **spend almost zero time in the app** while effortlessly documenting and executing your goals in real time.

---

## Installation

**macOS & Linux:**
```bash
curl -fsSL https://api.iepok.com/imp/install | sh
```

**Windows (PowerShell):**
```powershell
irm https://api.iepok.com/imp/install.ps1 | iex
```

---

## Commands

- `imp "action"` — Log an action  
```bash
  imp "5 pushups"
  imp "10 100m sprints"
```

- `imp plan "goal"` — Define actionable goals  
```bash
  imp plan "10 pullups every day"
```

- `imp analyze` — Get suggestions based on your actions

- `imp view` — Visualize progress and stats

---

## Philosophy

1. **Minimal Input, Maximal Output** — Quick logging, automatic analysis
2. **Implementation-Centric** — Only actionable tasks matter
3. **Real-Time Documentation** — Nothing is lost, everything is structured
